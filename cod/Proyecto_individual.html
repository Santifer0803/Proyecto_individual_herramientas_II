Ejemplo de los métodos de la clase¶

Para la creacin de los mtodos de la clase EditorExcel se usaron bastantes ejemplos, inicialmente se busc sobre cmo pasar un DataFrame de pandas a este formato, con el objetivo de que cualquier persona con conocimientos sobre DataFrames en python pueda usar la clase de manera sencilla. Los mtodos consisten en:

  • escribir(), para escribir texto, nmeros y frmulas simples (operaciones matemticas que no ocupen funciones propias de excel) en el Excel.
  • formula() permite escribir todas las frmulas de Excel que estn disponibles en la librera, sin importar el idioma en el cual el usuario tenga la computadora o Excel, las frmulas deben ponerse en ingls, pues as fu diseada la librera.
  • escribir_fila() se usa para pasar una lista de python a una fila de Excel.
  • escribir_columna() se usa para pasar una lista de python a una columna de Excel.
  • condicional() permite poner el fondo rojo a aquellas casillas que cumplan una condicin.
  • mini_barras() crea un grfico de barras pequeo, diseado para un anlisis rpido.
  • filtro() pone el formato de autofiltro en el rango seleccionado.
  • proteger() permite proteger el documento a travs de una contrasea, esto para que el usuario no pueda eliminar filas, columnas o modificar el texto (a elegir entre estas 3 opciones).
  • autoajuste() autoajusta todas las columnas que tengan, al menos, una observacin con datos.
  • cerrar(), para cerrar el editor de python.

Con los mtodos ya explicados, se procede a hacer un ejemplo prctico simulando situaciones que podran presentarse en el trabajo, para esto se crear un DataFrame pequeo en el cul se aplicarn los diversos mtodos implementados.

def elementos_simples(sopa_bs4, tipo, clase):
    '''
    Funcin que extrae los elementos que no ocupan modificaciones de un HTML

    Parameters
    sopa : bs4.BeautifulSoup
        HTML ledo a trevs de la librera BeautifulSoup
    tipo : str
        Tipo de elemento a buscar y extraer de un elemento 'sopa' del paquete BeautifulSoup
    clase : str
        Clase de elemento a buscar y extraer de un elemento 'sopa' del paquete BeautifulSoup

    Returns
    lista_simple : list
        Lista con los elementos encontrados dentro del HTML correspondiente
    '''
    # Lista para guardar los elementos correspondientes
    lista_simple = []

    # Se itera por cada uno de los elementos del HTML con la clase y el tipo especificado
    for fila in sopa_bs4.find_all(tipo, class_ = clase):
        # Se extrae el elemento y se agrega a la lista
        elemento = fila.get_text()
        lista_simple.append(elemento)

        # Se deja un espacio de un segundo entre cada iteracin, esto para no sobrecargar la pgina, pues en cada iteracin se mete a ella
        time.sleep(1)
    return lista_simple

def elemento_precio(sopa_bs4, tipo, clase):
    '''
    Funcin que extrae el precio, en colones, de cada carro de la pgina web de crautos

    Parameters
    sopa : bs4.BeautifulSoup
        HTML ledo a trevs de la librera BeautifulSoup
    tipo : str
        Tipo de elemento a buscar y extraer de un elemento 'sopa' del paquete BeautifulSoup
    clase : str
        Clase de elemento a buscar y extraer de un elemento 'sopa' del paquete BeautifulSoup

    Returns
    lista_precios : list
        Lista con los precios, en colones, de cada carro encontrados dentro de la pgina web de crautos
    '''
    # Se inicia extrayendo el tipo de cambio
    tipo_cambio = 0
    
    for fila in sopa_bs4.find_all('div', class_ = 'brandtitle2'):
        if(fila.get_text().count("Tipo de cambio: ") > 0):
            tipo_cambio = float(fila.get_text().split("")[1][0:5])
    
    # Lista para guardar los elementos correspondientes
    lista_precios = []

    # Se itera por cada uno de los elementos del HTML con la clase y el tipo especificado
    for fila in sopa_bs4.find_all(tipo, class_ = clase):
        # Removemos las comas de los precios
        fila_limpia = fila.get_text().replace(",", "")
        
        # Ponemos el precio, en caso de que est en dlares se multiplica por el tipo de cambio de la pgina
        if(fila_limpia.count("") > 0):
            coste = float(fila_limpia.split(" ")[1][0:8])
        elif(fila_limpia.count("$") > 0):
            coste = float(fila_limpia.split("$")[1][0:5])
            coste = coste * tipo_cambio
        else:
            coste = np.nan
        lista_precios.append(coste)

        # Se deja un espacio de un segundo entre cada iteracin, esto para no sobrecargar la pgina, pues en cada iteracin se mete a ella
        time.sleep(1)
    return lista_precios

def precio_final(sopa_bs4):
    '''
    Funcin que extrae el precio, en colones, de cada carro de la pgina web de crautos

    Parameters
    sopa : bs4.BeautifulSoup
        HTML ledo a trevs de la librera BeautifulSoup
    tipo : str
        Tipo de elemento a buscar y extraer de un elemento 'sopa' del paquete BeautifulSoup
    clase : str
        Clase de elemento a buscar y extraer de un elemento 'sopa' del paquete BeautifulSoup

    Returns
    lista_precios : list
        Lista con los precios, en colones, de cada carro encontrados dentro de la pgina web de crautos
    '''
    # Se inicia extrayendo el tipo de cambio de la pgina de crautos
    url = "https://crautos.com/autosusados/"
    
    # Manda una solicitud para obtener la informacin de la pgina web (librera requests)
    respuesta = requests.get(url)
    
    # Se analiza la informacin del HTML (librera BeautifulSoup)
    sopa_tc = BeautifulSoup(respuesta.content, 'html.parser')
    
    tipo_cambio = 0
    
    for fila in sopa_tc.find_all('div', class_ = 'brandtitle2'):
        if(fila.get_text().count("Tipo de cambio: ") > 0):
            tipo_cambio = float(fila.get_text().split("")[1][0:5])
    
    # Lista para guardar los elementos correspondientes
    lista_precios = []
    
    # Se extraen el monto y la moneda por separado
    moneda = elementos_simples(sopa_bs4, 'span', 'andes-money-amount__currency-symbol')
    monto = elementos_simples(sopa_bs4, 'span', 'andes-money-amount__fraction')
    
    # Se itera por cada uno de los elementos del HTML con la clase y el tipo especificado
    for iterador in range(len(monto)):
        # Removemos las comas de los precios
        cantidad = monto[iterador].replace(",", "")

        # Separamos los casos
        if(moneda[iterador].count("") > 0):
            coste = float(cantidad)
        elif(moneda[iterador].count("US$") > 0):
            coste = float(cantidad)
            coste = coste * tipo_cambio
        else:
            coste = np.nan
        lista_precios.append(coste)

        # Se deja un espacio de un segundo entre cada iteracin, esto para no sobrecargar la pgina, pues en cada iteracin se mete a ella
        time.sleep(1)
    return lista_precios
